Placement Committee Accountability Platform

AI-assisted tracking, reminders, and weekly assessments for internal accountability — with a public dashboard that shares only safe, aggregate outcomes.

How it works

  1. Members log outreach tasks and outcomes.
  2. AI agent monitors activity 24x7 and sends reminders.
  3. System evaluates weekly progress and suggests improvements.
  4. Public dashboard publishes non-sensitive aggregate metrics.

Internal metrics tracked

  • Unique companies reached
  • Leads generated and progressed
  • Average response time for assigned work
  • Average task completion time
  • On-time completion rate
